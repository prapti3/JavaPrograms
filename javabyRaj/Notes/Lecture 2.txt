Programming Concepts 
--------------------

Program
-------
A program is a collection of statements used to do a specific task

Comments
--------
Comments are used to document a program and ignored by programming language


Types of comments
----------------
1. Single line  //
2. Multi line   / *_____*/





Keywords
--------
A keyword is a reserved word/ predefined word whose meaning is known to the language directly 

    ex: int, float, string, if, else, char, while, do, for, class, public, private


Variable 
--------
 A Variable is an identifier which allocates some memory space
ex: empNum, studentName, grossSalary, itemPrice



Datatype 
---------
Type of data the variable holds is called datatype
ex: int
    char
    float
    String => Collection of characters


Declaration of Variable
-----------------------
In order to allocation memory space for variables we need to declare the variable


Syntax 
-------
datatype var1, var2, varn;
ex: 
   int a, b;
   char c;
   float f;
   String name;



Method/ Function 
----------------
A method is a subprogram which is used to do a specific task

EX: 
  // defining method
   int areaRect(int len, int breadth) {
         
         int area;
         area = len*breadth;
         return area;

   } 

   // calling method
    int a = areaRect(4,5);
    print(a);



   void areaRect(int length, int breadth){
       int area;
       area = length* breadth;
       print(area);
    }
   
   areaRect(4,5);

=> void means no Return type
